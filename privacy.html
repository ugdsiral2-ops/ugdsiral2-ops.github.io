GoogleCalOverlay 개인정보처리방침

최종 업데이트: 2026년 3월 22일

1. 수집하는 정보

GoogleCalOverlay는 Google OAuth 2.0을 통해 사용자의 Google 캘린더에 접근합니다. 수집되는 정보는 다음과 같습니다:

  • Google 캘린더 일정 (읽기 및 쓰기)
  • Google 계정 인증 토큰 (로컬 기기에만 저장)

2. 정보의 이용 목적

수집된 정보는 오직 다음 목적으로만 사용됩니다:

  • 사용자의 Google 캘린더 일정을 화면에 표시
  • 새로운 일정 추가

3. 정보의 저장 및 보안

GoogleCalOverlay는 서버를 운영하지 않습니다. 모든 데이터(인증 토큰 포함)는 사용자의 로컬 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

4. 제3자 정보 제공

수집된 정보는 어떠한 제3자에게도 제공, 판매, 공유되지 않습니다.

5. Google API 사용

본 앱은 Google Calendar API를 사용하며, Google 개인정보처리방침 및 Google API 서비스 사용자 데이터 정책을 준수합니다.

6. 접근 권한 철회

언제든지 Google 계정 권한 페이지에서 앱의 접근 권한을 철회할 수 있습니다.
